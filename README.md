Cli interface to Flipper ESP32 (WIFI Marauder) 

Works on Linux and MacOS

Usage: 
ff [/dev/ttyusbxxx] 

Don't forget to change the Flipper Zero's speed to 115200 (GPIO->USB-UART Bridge-Config->Baudrate 115200)
